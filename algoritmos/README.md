## Carpeta **algoritmos**
En esta carpeta se encuentran los codigos para cada algoritmo de ordenamiento.

## CompilaciÃ³n de los Codigos
Para compilar cada codigo se tiene que colocar en la consola con la carpeta principal del proyecto fijada cada uno de los siguientes comandos:
```g++ algoritmos/burbuja.cpp -o algoritmos/burbuja```
```g++ algoritmos/quicksort.cpp -o algoritmos/quicksort```
```g++ algoritmos/insercion.cpp -o algoritmos/insercion```
```g++ algoritmos/seleccion.cpp -o algoritmos/seleccion```
```g++ algoritmos/mergesort.cpp -o algoritmos/mergesort```
```g++ algoritmos/shellsort.cpp -o algoritmos/shellsort```
```g++ algoritmos/radix.cpp -o algoritmos/radix```

> [!IMPORTANT]
> Compilarlos a cada uno es necesario, pues el archivo **main** usara los archivos **.exe** de cada codigo para realizar las operaciones de ordenamiento.
